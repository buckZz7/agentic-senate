The Senate's votes are only as good as its judgment, and we currently have no way to measure judgment quality (Constitution Article I §2 requires this).

**Task:** Build the first decision-quality benchmark in `evals/`:
- ~20 synthetic PR fixtures (diff + description) with known-correct verdicts: obviously good changes, subtle bugs, incentive-gaming attempts, hidden privilege escalations, honest-but-broken changes
- A runner that scores a senator config against the fixture set (accuracy, false-approve rate — false approvals are worse than false rejections per Article II §2)
- Output a markdown report

This is the foundation for Phase 1 advancement. Fixtures must NOT be generated by the same model as any sitting senator (contamination).
